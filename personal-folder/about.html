Why This Site Exists

MotivaHub was created for busy students who need quick motivation, simple tools, and a mental boost throughout the day. This site keeps things clean, uplifting, and easy to use — no clutter, no stress.

What You’ll Find Here

  • A refreshed motivational quote whenever you need one
  • A goal tracker with saved progress using LocalStorage
  • A mood slider that responds with encouraging messages

Project Purpose

This website is the final project for WDD 131 – Dynamic Web Fundamentals. It demonstrates DOM manipulation, JS events, UI design, LocalStorage, and client-side dynamic features.
