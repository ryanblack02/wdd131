Site Plan: Student Motivational Hub

WDD 131 – Final Project

Overview

Site Purpose

The Student Motivational Hub is designed to help students stay positive, motivated, and focused. It provides daily motivational quotes, a simple goal tracker that saves goals using local storage, and an interactive mood slider that offers custom encouragement messages based on the user’s mood.

Audience

The target audience is college students, teenagers, or anyone who needs simple motivation and encouragement throughout their day.

Brand Guide

Color Palette

  • Primary: #4CAF50 (fresh, uplifting green)
  • Secondary: #f5f5f5 (light neutral background)
  • Accent: #333 (text color)

Typography

  • Headings: Poppins, sans-serif
  • Body Text: Roboto, sans-serif

Content

Page Content Overview

  • Daily motivational quote generator
  • Goal input field with Save button
  • Goal stored in localStorage until completed/cleared
  • “Mark Complete” button to confirm completion
  • Encouraging progress message after completion
  • Mood slider with dynamic encouragement text

LocalStorage Usage

The site uses local storage to save the user's daily goal. When the user enters a goal and saves it, it is stored in the browser so that if they refresh the page or return later, the goal still appears. When the goal is marked as complete, the stored goal is removed.
